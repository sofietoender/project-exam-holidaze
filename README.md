# Holidaze

A modern accommodation booking platform built with Next.js, React, and TypeScript for Noroff Project Exam 2.

## Description

Holidaze is a front-end application for an accommodation booking site. Users can browse and book venues, while venue managers can create and manage their properties.

## Built With

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Zustand (state management)
- Vitest (testing)

## Getting Started

### Installing

1. Clone the repo:

```bash
git clone git clone https://github.com/sofietoender/project-exam-holidaze.git
```

2. Install dependencies:

```bash
npm install
```

### Running

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Testing

```bash
npm test
```
